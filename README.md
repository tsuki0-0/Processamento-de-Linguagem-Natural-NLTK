# 🧠 Análise de Sentimentos e Processamento de Linguagem Natural (NLTK)

Aplicação desenvolvida em **Python**, utilizando **Streamlit** e **NLTK (Natural Language Toolkit)** para demonstrar conceitos fundamentais de **Processamento de Linguagem Natural (NLP)**.

O sistema permite analisar textos em português, aplicando técnicas de tokenização, frequência de palavras, remoção de stopwords, classificação por regras e análise de sentimentos.

---

## 📖 Sobre o Projeto

Este projeto reúne, em uma única aplicação, os principais conceitos introdutórios de NLP apresentados durante a disciplina, proporcionando uma demonstração prática das funcionalidades da biblioteca NLTK.

---

## 🚀 Funcionalidades

- 🔹 Tokenização de texto
- 🔹 Contagem da frequência das palavras
- 🔹 Identificação de mensagens prioritárias
- 🔹 Remoção de stopwords
- 🔹 Classificação simples de sentimentos
- 🔹 Direcionamento de mensagens por palavras-chave
- 🔹 Exibição das palavras mais frequentes
- 🔹 Classificação de mensagens por categoria
- 🔹 Limpeza e normalização de texto
- 🔹 Análise de sentimentos utilizando VADER
